## Lista de usuarios y productos
#### Obtener productos
```http
  GET /api/products
```
#### Obtener productos con id
```http
  GET /api/products/:pid?
```
#### Actualizar productos
```http
  PUT /api/products/:pid?
```
#### borrar productos
```http
  DELETE /api/products/:pid?
```
#### Crear productos
```http
  POST /api/products/
```
#### Obtener usuarios
```http
  GET /api/users
```

#### Obtener usuarios con id
